Which blastomere is α? Sperm entry vs. the other methods

Each column is a 2-cell embryo; each row is a method that labels one blastomere alpha and the other beta. A cell shows which original blastomere that method calls alpha — A = segmentation labels 1+3, B = labels 2+4. The alpha/beta direction is arbitrary per method, so use the row ⇄ buttons (or Auto-align) to flip rows until the columns agree — an aligned column is a consensus call for that embryo.

Rows marked ≈ are best-guess reconstructions of Harry's methods: his deck names each method and shows its results, but not the exact parameters (gene-panel membership, the per-gene t-statistic, the PCA axes), so these approximate the most likely implementation. Sperm entry and Higher total transcript are exact (the latter matches Harry's grids 20/20).
